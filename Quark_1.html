About Quark

A next-generation blockchain system designed to withstand quantum computing attacks.

Project Goals

Quark addresses the looming threat of quantum computers to current cryptographic systems. By implementing post-quantum cryptographic algorithms, we ensure blockchain security remains intact even in the quantum era.

Key Features

  • • Quantum-resistant digital signatures
  • • Secure wallet system with PQC encryption
  • • Optimized transaction processing
  • • Comprehensive benchmarking tools

PQC Algorithm Integration

SPHINCS+ (Digital Signatures)

  • • Lattice-based signature scheme
  • • NIST standardized algorithm
  • • Used for transaction signing
  • • Wallet authentication

FrodoKEM (Key Exchange)

  • • Learning with Errors (LWE) based
  • • Conservative security approach
  • • Wallet key generation
  • • Secure communication channels
